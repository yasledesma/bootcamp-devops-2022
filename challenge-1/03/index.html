Examen Git y Github

Alumna: Yasmin Ledesma

  1. Que es un sistema de control de versiones?

    Es un sistema que mantiene un registro de cada uno de los cambios que se realizan sobre los archivos de un repositorio.

  2. Cual es la diferencia entre git y github?

    Git es un sistema de control de versiones, que se utiliza para hacer track de los cambios a archivos, y Github es un servicio en la nube que proveé el almacenamiento de estos registros de control y de archivos.

  3. Cómo creo una rama en git?

    Dentro de un repositorio inicializado con git, se tiene que correr el comando git branch [nombre de la rama] en una terminal.

  4. Cómo clono un repositorio?

    Se puede hacer de dos maneras: La primera forma consiste en correr el comando git clone [link del repositorio], donde el link puede utilizar los métodos HTTP o SSH para descargar el repositorio. La segunda forma consiste en descargar el repositorio comprimido en formato .zip.
